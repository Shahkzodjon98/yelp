## Run Locally

Clone the project

bash
  git clone git remote add origin https://github.com/Shahkzodjon98/yelp-master.git

Go to the project directory

bash
  cd yelp-master

Install dependencies

bash
  npm install 

Start the server

bash
  npm run start
